## Commands

### /ryangif

Sends a random gif from ryan's saved gifs

### /ask

Ask mitsuri a question and see what she says

### /hi

Sends a greeting

### /prompt

Sends a random art prompt

#### Allowed Choices

- Characters
- Animals
- Situations
- Objects
